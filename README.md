# Retail Sales Analysis

## Dataset
Superstore Dataset

## Features
- Data cleaning and preprocessing  
- Sales trend analysis  
- Category and region-wise performance analysis  
- Top products identification  
- Sales prediction using linear regression  
- Visualization of insights and conclusions  

## Requirements
- Python 
- pandas  
- matplotlib  
- seaborn  
- scikit-learn  

## Outcome
- Understand sales patterns and trends  
- Identify key factors influencing sales  
- Apply data analysis and basic prediction techniques in a retail context
